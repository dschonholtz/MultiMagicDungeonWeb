## What is this?

MultiMagicDungeon Web is a browser-based port of the [MultiMagicDungeon](https://github.com/dschonholtz/MultiMagicDungeon) UE5 project. It is a multiplayer dungeon raider where:

- Players build their own dungeons that others raid
- Combat uses composable spell primitives (Fireball, Frostbolt, Telekinesis)
- The overworld connects all players' dungeons
- No install required — runs in any modern browser

This version participates in **Vibe Jam 2026** via the portal webring.

## How to Run

```bash
# Option 1: Just open the file
open index.html

# Option 2: Vite dev server (recommended)
npx vite
# Open http://localhost:5173
```

**Controls:**
- `WASD` — Move
- `Mouse` — Look around
- `Left Click` — Fireball
- `Right Click` — Frostbolt
- `E` — Telekinesis
- Walk into the green portal — Enter the Vibe Jam webring
